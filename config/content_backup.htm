快捷指令介紹

幾何

S 直線
R 矩形

完整的前置操作流程

第一步驟先新增三個終端機

第二步驟打開一個終端機，然後找到檔案init_replit的第七行

pip install flask flask_cors bs4 lxml pelican markdown gevent

貼到終端機裡讓終端機運行後打python main.py就可以產生出網站

之後就可以再網站上更改內容更改完之後要先處存再從動態轉到靜態，

後面回到程式打開第二個終端機打python main2,py確認更改的東西是否有在靜態

確認完之後打開第三終端機先打指令

git add .
git commit -m "打更改的內容"

最後打git push

git push


再push前要先檢查Pagrs裡面是否有更改到main

更改完就可以push

最後就可以檢查是否有成功運行
